# AI-Powered Multi-Modal Content Summarizer

An intelligent content analysis and summarization system built using Python, Streamlit, and BART.

This project extracts, analyzes, summarizes, translates, and converts content from multiple input sources such as text, documents, images, and video transcripts.

---

## Features ✨

* 📄 Document summarization (PDF, DOCX, TXT)
* 💬 Direct text summarization
* 📺 YouTube transcript extraction and analysis
* 🖼️ OCR-based text extraction from images
* 🌍 Multi-language translation
* 🔊 Text-to-speech output
* 🤖 Transformer-based AI summarization

---

## Installation 🚀

Clone the repository:

git clone <your-github-repo-link>

Move into project folder:

cd ex_sum

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
